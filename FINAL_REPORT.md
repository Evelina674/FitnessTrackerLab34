# Final Report

## Опис проєкту

FitnessTracker — це консольний застосунок для ведення тренувань та аналізу фізичної активності користувача.

Проєкт дозволяє:

- створювати тренувальні сесії;
- працювати з cardio та strength активностями;
- переглядати статистику;
- зберігати та відновлювати дані з JSON-файлу;
- виконувати аналітичні LINQ-запити.

---

## Архітектура

Проєкт побудований за багатошаровою архітектурою:

- Domain

- Console

Основні принципи:

- Separation of Concerns
- Dependency Inversion

## Використані патерни

### Repository Pattern

Відповідає за доступ до даних та ізоляцію бізнес-логіки від сховища.

### Factory Pattern

Використовується для створення різних типів активностей через єдиний механізм.
