# Release Plan v1.0.0

## Що входить у версію v1.0.0

До релізу v1.0.0 входить:

- консольний застосунок FitnessTracker;
- створення профілю користувача;
- створення тренувальних сесій;
- кардіо активності;
- силові активності;
- JSON persistence;
- SaveAsync та LoadAsync;
- InMemory repository;
- JSON repository;
- Factory Pattern;
- LINQ-аналітика;
- unit tests;
- integration tests;
- fault handling tests;
- coverage support;
- GitHub Actions CI;
- README, TESTING та технічна документація.

---

## Що переноситься після курсу

Після завершення курсу можна додати:

- повноцінний веб-інтерфейс;
- авторизацію користувачів;
- базу даних замість JSON-файлу;
- графіки прогресу;
- експорт у PDF або CSV;
- синхронізацію між пристроями;
- більш розширену аналітику.

---

## Допустимі технічні борги

На момент релізу залишаються такі допустимі обмеження:

- консольний інтерфейс замість GUI або web UI;
- JSON-файл замість бази даних;
- відсутність багатокористувацької авторизації;
- базова аналітика без графіків;
- обмежене логування помилок.
