# Iteration 2 Plan

## Сценарії для реалізації

У другій ітерації будуть реалізовані:

- створення тренувальної сесії;
- збереження тренувань у JSON-файл;
- завантаження тренувань із JSON-файлу;
- перегляд тренувань користувача;
- аналітика тренувань за категоріями;
- пошук і сортування тренувань.

## Класи без змін

Без значних змін залишаються:

- Activity
- CardioActivity
- StrengthActivity
- Distance
- ActivityRecord

## Точки розширення

Планується використання:

- Repository Pattern;
- Factory Pattern;
- JSON Persistence Layer;
- LINQ Analytics Service.

## Потенційні ризики

- дублювання логіки перевірок;
- помилки серіалізації JSON;
- складність інтеграційного тестування;
- зростання складності консольного меню;
- залежність між сервісами та репозиторіями.
