# User Guide

## Про застосунок

FitnessTracker — це консольний застосунок для відстеження тренувань та активностей користувача.

## Запуск програми

Виконайте команду:

```bash
dotnet run --project src/FitnessTracker.Console
```

## Створення профілю

Після запуску введіть:

- ім'я;
- вік;
- вагу;
- зріст.

## Головне меню

1. Add workout session
2. Show session summary
3. Save sessions to JSON
4. Load sessions from JSON
5. Show analytics
6. Exit

## Додавання тренування

1. Оберіть пункт 1.
2. Введіть дату.
3. Введіть назву активності.
4. Оберіть тип:
   - cardio
   - strength

## Cardio Activity

Для cardio введіть:

- назву;
- тривалість;
- дистанцію;
- нотатки.

## Strength Activity

Для strength введіть:

- назву;
- тривалість;
- підходи;
- повторення;
- вагу;
- нотатки.

## Збереження даних

Використайте:

3. Save sessions to JSON

## Завантаження даних

Використайте:

4. Load sessions from JSON

## Перегляд статистики

Використайте:

2. Show session summary

або

5. Show analytics

## Обробка помилок

Програма повідомляє про:

- неправильні дати;
- порожні активності;
- неправильний тип активності;
- від'ємні значення;
- неправильні підходи та повторення.
